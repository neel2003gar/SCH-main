GLOBAL RULES FOR BOTH PAGES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. NO MAP anywhere on either page. The province 
   accordion IS the location finding tool.

2. Both pages use the same sticky nav and footer 
   as all other pages in the design.

3. The "Locations" nav dropdown links to both pages:
   → Find a Rehabilitation Clinic
   → Find an Occupational Health Clinic

4. EN/FR toggle switches all text to French:
   "Find a Rehabilitation Clinic" → 
     "Trouver une clinique de réadaptation"
   "Find an Occupational Health Clinic" → 
     "Trouver une clinique de santé au travail"
   "Let Us Help You Get Back to Work" → 
     "Laissez-nous vous aider à reprendre le travail"
   Province names stay in English/French as applicable 
   (Ontario stays Ontario, Québec stays Québec)

5. Accordion animation: smooth expand/collapse with 
   200ms ease transition. City pills fade in when 
   province expands.

6. On mobile: accordion collapses to full-width rows, 
   city pills stack to 2-column grid instead of 3.

7. Color system (same as rest of site):
   Navy bg: #0F2A45
   Teal accent: #1D9E75
   Green: #6ABF4B
   Section bg light: #F4F4F2
   Card bg: #FFFFFF
   Border: #EEEDE8
   Heading text: #1A3A5C
   Body text: #333333
   Subtext: #777777

8. Accessibility widget (navy circle, bottom-left 
   fixed) present on both pages as on all other pages.

9. Both pages link from the Locations nav dropdown 
   AND from the Locations hub page (parent page).
