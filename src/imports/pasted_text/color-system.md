GOLDEN RULES — NEVER BREAK THESE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. #6ABF4B is ONLY used for:
   Primary buttons | Section labels | Step numbers |
   Certifications | Accent dots | Footer "Get a Quote" link |
   Tag pills on dark bg
   NEVER use #6ABF4B as body text color

2. #2B7DC8 is ONLY used for:
   Text links | Icon colors | Active nav states |
   Active language toggle | Focus borders on inputs |
   Specialty labels under team names
   NEVER use #2B7DC8 as a button background

3. #1A3A5C is ONLY used for:
   Dark section backgrounds | Hero backgrounds | Footer |
   All H1/H2/H3 headings | Logo text | Outline button borders
   NEVER mix with #155F8A in the same section

4. Background section order — always alternate:
   DARK (#1A3A5C) → LIGHT → DARK → LIGHT
   Never two dark navy sections in a row
   Never two #EAF4FB sections in a row

5. Text on dark (#1A3A5C) backgrounds:
   ALWAYS white. Never use #333333 or #1A3A5C text on navy bg

6. Card borders are ALWAYS #EEEDE8 at 1px
   NEVER use thick borders except for accent left borders (4px)
   and active states (1.5px)

7. Border radius:
   Large cards + hero overlays: 16px
   Buttons + small cards:       8–10px
   Input fields:                8px
   Tag pills + small badges:    20px (fully rounded)
   Icon circles:                50% (fully round)

8. The ONLY colors allowed as section backgrounds are:
   #FFFFFF | #EAF4FB | #F4F4F2 | #F0F7F0 | #1A3A5C | #155F8A
   No other colors as section backgrounds

9. Box shadows are subtle only:
   Cards: rgba(0,0,0,0.06) at 0 4px 16px
   Hover: rgba(43,125,200,0.10) at 0 6px 20px
   Never black shadows, never heavy shadows

10. Form inputs always:
    Default border: #EEEDE8
    Focus border: #2B7DC8
    Error border: #E24B4A
    Success border: #6ABF4B
